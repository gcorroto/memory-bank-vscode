# Changelog

All notable changes to Memory Bank Inspector will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.0.47] - 2026-01-24

### Added
- Soporte para mostrar Keywords y Responsibilities en la vista de Agentes Activos (sincronizado desde SQLite).
- Nueva sección para visualizar Orchestrator Logs en la actividad de agentes.
- Mejores interfaces de datos para sincronización con memory-bank-mcp.

## [0.0.1] - 2026-01-11

### Added
- Initial public release as Memory Bank Inspector
- Project explorer for Memory Bank indexed projects
- Indexed files tree view with hierarchical folder structure
- Project documentation viewer with Markdown preview
- Code Relations analyzer with React Flow visualization
- AI-enriched node descriptions (Spanish)
- Automatic graph layout using dagre algorithm
- Node and edge highlighting for dependency tracking
- Project deletion with LanceDB embeddings cleanup
- Integration with Memory Bank MCP

### Technical
- LanceDB integration for embeddings management
- OpenAI API integration (GPT-5.x models)
- React Flow for dataflow visualization
- Webpack bundling for webviews
